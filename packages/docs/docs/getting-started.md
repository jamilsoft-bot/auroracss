# Getting Started

## Install
```bash
pnpm add @aurora/core @aurora/tokens @aurora/react
```

## Include CSS
```css
@import '@aurora/core/dist/aurora.css';
@import '@aurora/tokens/tokens.css';
```

## React usage
```tsx
import { Button } from '@aurora/react';

export const Example = () => <Button>Launch Aurora</Button>;
```
